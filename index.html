스프레드시트 연동 방법

아래는 스프레드시트를 웹페이지에 표시한 예시입니다. 실제 연동을 위해서는 다음 단계를 따르세요:

  1. 구글 스프레드시트 생성 (열: 타임스탬프, 서비스, 아이디, 비밀번호, 메모)
  2. 파일 > 웹에 게시 > 시트 선택 > 웹 페이지로 게시
  3. 생성된 임베드 링크를 아래 iframe의 src에 붙여넣기

👇 여기에 게시된 스프레드시트가 표시됩니다 👇

스프레드시트 연동 상세 가이드

1. 구글 폼으로 연동하기

가장 쉬운 방법은 구글 폼을 사용하는 것입니다:

  1. 구글 폼(Google Forms)에서 새 폼 생성
  2. 필요한 필드 추가 (서비스명, 아이디, 비밀번호, 메모)
  3. 응답 탭에서 스프레드시트 연결 선택
  4. 폼 공유 링크를 받아 웹사이트에 임베드

구글 폼 HTML 코드 예시:
<iframe src="https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform?embedded=true" width="100%" height="500" frameborder="0" marginheight="0" marginwidth="0">로드 중...</iframe>

2. Google Apps Script 사용하기

더 많은 제어와 기능이 필요하다면 Google Apps Script를 사용하세요:

2-1. 스프레드시트에 스크립트 추가하기

스프레드시트 메뉴에서 '확장 프로그램 > Apps Script'를 선택하고 다음 코드를 입력:

function doGet() {
  return HtmlService.createHtmlOutputFromFile('Index')
    .setTitle('네이버 계정 관리 시스템');
}

function addAccount(data) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  sheet.appendRow([new Date(), data.service, data.userId, data.password, data.memo]);
  return "계정이 성공적으로 저장되었습니다!";
}

function getAccounts() {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = sheet.getDataRange().getValues();
  return data;
}

2-2. HTML 파일 생성하기

'파일 > 새로 만들기 > HTML 파일'을 선택하고 이름을 'Index'로 지정한 후 HTML 코드 작성

2-3. 웹 앱으로 배포하기

'배포 > 새 배포'를 클릭하고 유형을 '웹 앱'으로 선택한 후 배포

⚠️ 보안 주의사항

이 예시는 교육 목적으로만 사용하세요. 실제 비밀번호를 평문으로 저장하는 것은 매우 위험합니다.

  • 비밀번호는 항상 암호화하여 저장해야 합니다.
  • 중요한 계정 정보는 적절한 보안 조치가 있는 시스템을 사용하세요.
  • 공개 스프레드시트에 중요 정보를 저장하지 마세요.
  • 접근 권한을 제한하고 데이터 보호 정책을 준수하세요.
