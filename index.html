GitHub 사이트 연동 방법

1. GitHub 계정 만들기

아직 GitHub 계정이 없다면, github.com에서 무료 계정을 만듭니다.

2. 새 저장소(Repository) 만들기

  1. GitHub에 로그인 후, 오른쪽 상단의 '+' 버튼을 클릭하고 'New repository'를 선택합니다.
  2. 저장소 이름을 'mz-marketing-system'으로 입력합니다.
  3. 'Public' 옵션을 선택합니다.
  4. 'Add a README file' 옵션을 체크합니다.
  5. 'Create repository' 버튼을 클릭합니다.

3. 이 HTML 파일 업로드하기

  1. 생성된 저장소 페이지에서 'Add file' 버튼을 클릭하고 'Upload files'를 선택합니다.
  2. 이 HTML 파일을 'index.html'로 저장한 후 드래그 앤 드롭으로 업로드합니다.
  3. 'Commit changes' 버튼을 클릭합니다.

4. GitHub Pages 활성화하기

  1. 저장소 페이지에서 'Settings' 탭을 클릭합니다.
  2. 왼쪽 메뉴에서 'Pages'를 클릭합니다.
  3. 'Source' 섹션에서 'Branch: main'을 선택하고 저장합니다.
  4. 몇 분 후에 웹사이트가 생성되며, 표시되는 URL을 통해 접속할 수 있습니다.
  5. 보통 URL 형식은 'https://[사용자명].github.io/mz-marketing-system' 입니다.

구글 스프레드시트 연동 방법

1. 구글 스프레드시트 만들기

  1. Google Sheets에 접속하여 새 스프레드시트를 만듭니다.
  2. 여러 시트를 만들어 각각 '업체정보', '작업현황', '계정관리' 등으로 이름을 지정합니다.
  3. 각 시트의 첫 행에 열 제목을 입력합니다 (예: 업체명, 연락처, 계약일 등).

2. 구글 시트 공유 설정

  1. 스프레드시트 우측 상단의 '공유' 버튼을 클릭합니다.
  2. 팀원들의 이메일을 입력하여 편집 권한을 부여합니다.
  3. '고급' 옵션에서 '웹에 게시'를 선택하여 웹사이트에서 데이터를 읽을 수 있게 합니다.

3. 웹사이트와 연동하기

  1. 웹사이트 상단의 '구글 스프레드시트 연결' 버튼을 클릭합니다.
  2. Google 계정으로 로그인하고 스프레드시트에 대한 접근을 허용합니다.
  3. 연결 후에는 웹사이트에서 데이터를 읽고 쓸 수 있습니다.
  4. 변경사항을 저장하려면 '데이터 저장' 버튼을 클릭합니다.
